# Sound Files

Place the following MP3 files in this folder. The app gracefully handles missing files (silent fallback).

| File              | Purpose                                       | Suggestion                            |
|-------------------|-----------------------------------------------|---------------------------------------|
| `lofi-bg.mp3`     | Background lo-fi loop during Focus            | Lo-fi study music (loop-friendly)     |
| `tick.mp3`        | Tick-tock each second (toggle in Settings)    | Short clock tick                      |
| `bell.mp3`        | Plays when a session completes                | Bell, chime, ding                     |
| `click.mp3`       | UI click feedback                             | Short pixel/8-bit click               |
| `achievement.mp3` | Plays when an achievement unlocks             | Fanfare / level-up jingle             |
| `meow.mp3`        | Plays when you click the cat                  | Cat meow                              |

## Free sources
- [Pixabay Sounds](https://pixabay.com/sound-effects/) (no attribution needed)
- [Freesound](https://freesound.org/) (CC licensed)
- [Mixkit](https://mixkit.co/free-sound-effects/) (free)
- [Zapsplat](https://www.zapsplat.com/) (free with account)

## Quick start
1. Download any 6 mp3 files matching the names above.
2. Drop them into this `public/sounds/` folder.
3. Refresh the app — sounds will play automatically.

The app works fine without sound files; you'll just have a silent (but still cute) experience.
