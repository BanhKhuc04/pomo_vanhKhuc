# Avatar Assets Guide

Place your generated transparent PNG layers here.

**Rules:**
1. Files must be **PNG format** with **transparent background**.
2. **Canvas size:** 256x256 px.
3. **Alignment:** All layers must share the exact same anchor point and fit within the identical canvas space so they stack perfectly.
4. **Naming:** File names must strictly follow the convention defined in `src/data/avatarAssetChecklist.js`.

**Minimum Viable Product (MVP) Assets to Test Pipeline:**
- `base/idle.png`
- `base/typing.png`
- `base/relax.png`
- `base/celebrate.png`
- `hair/messy-black.png`
- `glasses/round.png`
- `outfit/hoodie-black.png`
- `accessory/headphones.png`
- `pets/cat-sleep.png`

For the full detailed documentation, please read `AVATAR_ASSET_SPEC.md` in the project root.
